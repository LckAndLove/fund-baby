# 养基场

养基场是一款基于 Tauri 2 的桌面基金持仓小工具，用于查看基金估值、持仓收益和桌面悬浮收益小条。

它只面向桌面端使用：主面板负责基金列表、持仓、收益统计和设置；隐藏后会显示一个轻量收益小条，适合长期放在桌面角落查看当日收益。

## 功能

- 基金估值：根据基金代码拉取单位净值、估算净值、涨跌幅和更新时间。
- 持仓管理：支持添加、删除、编辑基金持仓份额、成本、分组和备注。
- 收益统计：统计总金额、日收益、持有收益和收益率。
- 是否计入收益：基金可以保留显示，但不参与总收益计算。
- 桌面悬浮窗：支持透明度调节、置顶、拖动、托盘控制和隐藏显示。
- F2 快捷键：按 `F2` 在主面板和收益小条之间切换，恢复主面板时会自动刷新。
- 收益小条：隐藏主面板后显示当日收益，宽度随内容自适应，金额使用 `K/M` 格式并保留两位小数。
- 托盘菜单：支持显示、隐藏、开机自启动、透明度设置、导入配置、导出配置和退出。
- 自动刷新：默认每 10 秒刷新一次，也可以手动刷新。
- 配置持久化：基金列表、持仓、分组、折叠状态、刷新频率、视图状态等会保存在本地。

## 常用操作

- `F2`：主面板和收益小条切换。
- 托盘图标左键：显示主面板并刷新。
- 托盘菜单 `显示`：打开主面板。
- 托盘菜单 `隐藏到托盘`：隐藏主面板。
- 托盘菜单 `导入配置`：从本地 JSON 文件恢复配置。
- 托盘菜单 `导出配置`：把当前配置保存为 JSON 文件。
- 面板内刷新按钮：立即拉取最新行情。
- 编辑模式：维护基金、持仓金额、成本和是否计入收益。

收益小条默认显示在工作区左下角，可拖动调整位置。主面板和收益小条的位置分别记录，互不覆盖。

## 技术栈

- Tauri 2
- Next.js 16
- React 18
- ECharts
- dayjs
- Rust
- 原生 CSS

桌面壳由 Tauri 提供，界面层由 React 渲染。行情数据来自公开基金和行情接口，接口可用性与数据准确性受第三方服务影响。

## 本地环境

需要安装：

- Node.js `>=20.9.0`
- npm
- Rust stable
- Windows 上构建 Tauri 所需的系统依赖

安装依赖：

```bash
npm install
```

启动桌面开发模式：

```bash
npm run dev:desktop
```

## 构建桌面应用

构建 Windows 桌面程序：

```bash
npm run build:desktop
```

构建完成后的可执行文件：

```text
src-tauri/target/release/fund-baby.exe
```

安装包目录：

```text
src-tauri/target/release/bundle/nsis/
```

如果构建时报 `fund-baby.exe` 无法删除，通常是旧程序还在运行。先退出托盘程序，或结束旧进程后重新构建。

## 配置导入导出

导出的配置包含：

- 基金列表
- 持仓信息
- 分组信息
- 折叠状态
- 刷新频率
- 当前视图状态
- 待处理交易记录

导入配置会覆盖当前本地配置。操作前建议先导出一份备份。

## 数据与计算

日收益按当前持仓金额和基金当日涨跌幅估算。基金如果被设置为不计入收益，会继续显示行情，但不会参与总金额、日收益和持有收益统计。

行情和估值数据可能存在延迟、缺失或第三方接口不可用情况。盘中估值不是基金公司最终净值，实际收益以交易平台和基金公司披露为准。
